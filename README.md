## üèó System Architecture

The site is designed with a **Modular Component Architecture** and strict TypeScript interfaces, ensuring that the UI remains decoupled from the content logic. This allows for high maintainability, predictable data flow, and easy integration of new features or projects.

## üöÄ Getting Started

### Installation

1.  **Clone the repo:**

    ```bash
    git clone [https://github.com/HarshitK150/portfolio.git](https://github.com/HarshitK150/portfolio.git)
    cd portfolio
    ```

2.  **Install dependencies:**

    ```bash
    npm install
    ```

3.  **Run the development server:**
    ```bash
    npm run dev
    ```
    Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
